# Neutron Tools

[![CurseForge Downloads](https://img.shields.io/curseforge/dt/1232801?logo=curseforge)](https://www.curseforge.com/minecraft/mc-mods/neutron-tools)
[![Custom Badge](https://img.shields.io/badge/Download-CurseForge-orange?style=for-the-badge&logo=curseforge)](https://www.curseforge.com/minecraft/mc-mods/neutron-tools)

Provides unique utilities and commands not commonly found in other mods. This mod was made with the purpose of enhancing the development and control of your modpack.


# Current Features
- Edit the Creative Inventory!
- You can modify all creative tabs in any way you like! Remove tabs, add new tabs, replace tabs, order tabs, and edit items within tabs. Its quite easy to do, Refer to the wiki to learn how.
- Configurable Portal Wait Time!
- The mod has a config option to change the wait time for all portals in survival or adventure mode!
- Configurable Hunger speed!
- Disable experimental settings popup!
- Configurable hunger level!

## A few of the Commands:
See all commands on the wiki
- neutron listall command (op only)
-   Can be used to list all items, entities, blocks, creative tabs, creative tab items etc in your modpack
- neutron creativetabs command (op only)
-   Used to reload creative tabs, or display creative tab internal names
- neutron ping
-   Ping all players or one specific player and get network latency (Available to even Non OP players!)
